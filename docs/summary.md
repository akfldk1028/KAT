이게 뭔가요?

  카카오톡에서 사기 메시지를 잡아내는 AI 시스템입니다.

  누군가 "엄마 폰 고장났어 돈 보내줘" 같은 메시지를 보내면 → "이거 사기야!" 라고 경고해줍니다.     

  ---
  어떻게 작동하나요? (3단계)

  메시지 받음
      ↓
  [1단계] 블랙리스트 확인 (0.01초)
      → 이미 신고된 계좌/번호면 즉시 차단
      ↓
  [2단계] AI가 내용 분석 (0.05초) ⭐핵심
      → "이 메시지가 사기인지 아닌지" 맥락 파악
      → "엄마 생일 선물" = 정상
      → "엄마 폰 고장 돈 보내" = 사기
      ↓
  [3단계] AI가 최종 판단 (0.1초)
      → 위험도 + 왜 위험한지 설명

  ---
  어떤 사기를 잡나요? (9가지)

  | 유형  | 설명       | 예시                      |
  |-----|----------|-------------------------|
  | A-1 | 가족 사칭    | "엄마야 폰 고장 돈 보내줘"        |
  | A-2 | 상사/지인 사칭 | "팀장인데 급해서 100만원만"       |
  | A-3 | 상품권 요청   | "기프트카드 사서 번호 보내줘"       |
  | B-1 | 택배/청첩장   | "[택배] 주소 확인 bit.ly/xxx" |
  | B-2 | 기관 사칭    | "[검찰청] 계좌 동결됨"          |
  | B-3 | 결제 사기    | "[해외결제] 98만원 승인"        |
